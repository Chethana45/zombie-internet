Zombie Internet 🧟‍♂️🌐

A nostalgic full-stack web experience that recreates the vibe of old internet culture — trends, aesthetics, fake feeds, alerts, terminals, and chaotic digital nostalgia.

✨ Features
📰 Retro internet-style feed
🚨 Fake alert system
💻 Interactive terminal UI
📊 Dashboard components
🎨 Clean modern + nostalgic interface
⚡ Built with Next.js and TypeScript
🛠️ Tech Stack
Next.js
TypeScript
Tailwind CSS
React
PostCSS

🚀 Getting Started

Clone the repository:

git clone https://github.com/Chethana45/zombie-internet.git

Go to project folder:

cd zombie-internet

Install dependencies:

npm install

Run development server:

npm run dev

Open:

http://localhost:3000


“Zombie Internet” imagines a digital world where old internet culture never died — blending nostalgia, aesthetics, and chaotic online energy into a modern interactive experience.
